# 汉字学习乐园 🎯

一个交互式的汉字学习网站，帮助学生通过翻卡片的方式学习汉字的读音和组词。

## ✨ 功能特点

### 📚 核心功能
- **随机抽取学习**：每轮随机抽取10个不重复的汉字
- **翻卡片交互**：点击卡片翻转查看拼音和组词
- **语音朗读**：点击喇叭图标听取汉字发音（支持移动端）
- **两种学习模式**：
  - 普通模式：显示单字，学习拼音
  - 挑战模式：显示词语，反向学习汉字

### 📊 进度追踪
- **本轮进度**：显示当前轮次的学习进度（0-10个字）
- **总体进度**：统计已完成的不重复汉字数量（去重计算）
- **进度可视化**：渐变进度条实时展示学习成果

### ⭐ 收藏系统
- 收藏不认识的字，方便复习
- 查看收藏列表，点击直接复习
- 收藏数据持久化保存

### 🎨 设计亮点
- 复古风格配色（绿色主题）
- 卡通背景水印
- 流畅的翻转动画
- 响应式设计，适配移动端和PC端

## 📖 学习内容

- **汉字数量**：600+ 个常用汉字
- **内容来源**：三字童谣识字材料
- **包含信息**：汉字、拼音、常用组词

## 📱 移动端使用说明

### 声音播放
1. 首次进入页面，底部会显示"🔊 点击启用声音"提示
2. 点击提示或直接点击喇叭按钮启用音频
3. 确保手机没有开启静音模式

### 操作提示
- 点击卡片翻转查看拼音
- 点击喇叭图标听发音
- 左右滑动切换卡片（键盘方向键）
- 空格键快速翻转卡片

## 🎓 教学设计理念

### 认知负荷管理
- 每轮只学习10个字，避免信息过载
- 渐进式信息呈现（先汉字，后拼音，再组词）

### 即时反馈
- 翻转动画提供视觉反馈
- 语音播放提供听觉反馈
- 进度条实时更新

### 游戏化元素
- 完成一轮后的祝贺动画
- 收藏系统增加成就感
- 挑战模式提供难度选择
